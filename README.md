Long URLs are ugly, hard to share, and difficult to remember, 
So here I am with a URL SHORTENER, with core backend features. 
It solves the problem of long, unshareable URLs by generating unique short codes and storing them in PostgreSQL. When someone visits the short link, it redirects them to the original URL. I also added a click counter to track popularity. This taught me database design, API architecture, and handling real-world issues like unique code collisions.
